# Data Visualization Builder

Create interactive charts, dashboards, and data visualizations from CSV/JSON data using Chart.js, D3.js, matplotlib, and Plotly. Includes automated insights and chart recommendations.

## Features

- **Chart Types**: Line, bar, pie, scatter, heatmap, area, radar
- **Libraries**: Chart.js, D3.js, matplotlib, Plotly
- **Data Formats**: CSV, JSON, arrays, objects
- **Auto-Insights**: Trend detection, statistics
- **Responsive**: Mobile-friendly charts
- **Export**: PNG, SVG, HTML

## Installation

Available in Claude.ai, Claude Code, and Claude API.

## Usage

Ask Claude:
> "Create a bar chart from this CSV data: [paste data]"

Output includes:
1. **Chart Code** - Complete implementation
2. **HTML Template** - Ready-to-use page
3. **Insights** - Data analysis
4. **Recommendation** - Best chart type

## Best Practices

- Choose appropriate chart types
- Use clear labels and titles
- Include legends for multiple series
- Make charts responsive
- Use consistent color schemes
- Add tooltips for interactivity

## Chart Types

- **Line**: Time series, trends
- **Bar**: Categorical comparisons
- **Pie**: Proportions, percentages
- **Scatter**: Correlations
- **Heatmap**: Multi-dimensional patterns
- **Area**: Cumulative data
