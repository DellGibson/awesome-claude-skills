# Awesome Claude Skills - Documentación en Español

Bienvenido a la documentación en español de Awesome Claude Skills. Esta es una traducción profesional de las habilidades más utilizadas.

## Habilidades Traducidas

Esta documentación incluye traducciones al español de las 5 habilidades principales:

### 1. [Mejorador de Imágenes](./image-enhancer.md)
Mejora la calidad de imágenes y capturas de pantalla aumentando resolución, nitidez y claridad.

**Categoría:** Creative & Media
**Archivo Original:** [image-enhancer/README.md](../../../image-enhancer/README.md)

### 2. [Descargador de Videos](./video-downloader.md)
Descarga videos de YouTube y otras plataformas para visualización sin conexión, edición o archivo.

**Categoría:** Creative & Media
**Archivo Original:** [video-downloader/README.md](../../../video-downloader/README.md)

### 3. [Pruebas de Aplicaciones Web](./webapp-testing.md)
Kit de herramientas para probar aplicaciones web locales usando Playwright.

**Categoría:** Development & Code Tools
**Archivo Original:** [webapp-testing/README.md](../../../webapp-testing/README.md)

### 4. [PDF](./pdf.md)
Kit de herramientas integral para manipulación de PDF: extraer texto, crear documentos, fusionar/dividir y manejar formularios.

**Categoría:** Productivity & Organization
**Archivo Original:** [document-skills/pdf/README.md](../../../document-skills/pdf/README.md)

### 5. [Fábrica de Temas](./theme-factory.md)
Aplica temas profesionales de fuentes y colores a artefactos incluyendo diapositivas, documentos e informes.

**Categoría:** Creative & Media
**Archivo Original:** [theme-factory/README.md](../../../theme-factory/README.md)
